# Contributing to Arena FPS

Thanks for your interest in contributing! This is a browser-based FPS and contributions of all kinds are welcome.

---

## Getting Started

1. **Fork** the repository
2. **Clone** your fork: `git clone https://github.com/YOUR_USERNAME/arena-fps.git`
3. Open `index.html` in a browser via a local dev server (see README)
4. Make your changes
5. Submit a **Pull Request** with a clear description

---

## Code Style

- Plain JavaScript — no frameworks, no bundler
- Keep logic self-contained in `index.html` for simplicity
- Comment non-obvious sections
- Use `const` / `let`, no `var`
- Prefer descriptive variable names over single letters

---

## What to Work On

### Good First Issues
- Add more weapon types (e.g., LMG, revolver)
- Add sound effects using the Web Audio API
- Add floor markings / spawn indicators
- Improve enemy AI pathfinding
- Add a round timer / scoreboard HUD

### Larger Projects
- Multiplayer via WebSockets (e.g., Socket.io)
- Map editor / procedural generation
- Mobile touch controls
- Save/load loadout (localStorage)
- Grenade / utility system

---

## Submitting Changes

- Keep PRs focused — one feature or fix per PR
- Test in Chrome and Firefox before submitting
- Update `README.md` if you add new controls or weapons
- Don't commit minified or auto-generated files

---

## Reporting Bugs

Open an issue with:
- Browser and OS
- Steps to reproduce
- Expected vs actual behavior
- Screenshot or screen recording if helpful

---

## Questions?

Open a GitHub Discussion or an issue labeled `question`.
